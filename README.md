<h2>프로젝트 소개</h2>
<ul>
  <li>길잡이는 기업과 개인의 구인구직을 지원하며, 맞춤형 솔루션과 풍부한 취업 정보를 제공합니다.</li> 
  <li>이력서별 평점 기능, 개인 맞춤 알림 서비스, 추천 공고 및 이력서 목록, 커뮤니티 기능 등 회원들의 편리함과 자율성을 중시하여 다양한 서비스를 제공합니다.</li>
</ul>
